# Alpha Archives Blog

Express and EJS blog-style web application inspired by retro gaming communities.

## Features
- User signup/login
- Session authentication
- Blog post creation
- User profiles
- Dynamic EJS rendering
- Custom themed UI
- Responsive layouts

## Built With
- Node.js
- Express
- EJS
- CSS
- express-session

## Status
In development.
